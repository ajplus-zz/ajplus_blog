                      3）方法重载

                      ajplus小于 1 分钟javabasic

                      方法重载（Overloading）：在同一类内定义的一组或多组同名方法；

                      • 同一性：所有参与重载的方法均位于同一个类的内部定义；
                      • 名称一致性：这些方法共享相同的方法名称，但参数列表各不相同；
                      • 参数差异性；

                      需要注意的是：

                      • 重载的判断基于方法的声明和定义，与方法的实际调用方式无直接关联；无论调用时使用何种形式，编译器会根据传入参数自动匹配到对应的方法实现；
                      • 方法的返回类型并不作为判定重载的依据，即使两个方法的返回类型不同，但如果它们的参数列表相同，则不会构成重载；
